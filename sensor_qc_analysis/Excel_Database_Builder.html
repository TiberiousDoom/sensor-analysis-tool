ℹ️ Database Schema

The compiled database creates a sensor_readings table compatible with the Sensor Analysis Tool:

sensor_readings (
  id, Job #, Serial Number, Channel,
  0, 5, 15, 30, 60, 90, 120, Test #, Timestamp, Initials
)

Your Excel files should have columns matching these headers. The tool will automatically map common variations like "Job Number" → "Job #" and "Date" → "Timestamp".
