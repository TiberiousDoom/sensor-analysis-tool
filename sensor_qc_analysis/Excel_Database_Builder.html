ℹ️ Database Schema

The compiled database creates a sensor_readings table compatible with the Sensor Analysis Tool:

sensor_readings (
  id, Job #, Serial Number, Channel,
  0, 5, 15, 30, 60, 90, 120, Test #, Timestamp, Initials
)

Your Excel files should contain a sheet named "Test Results" or "Potted Test 1"–"Potted Test 4" with columns matching these headers. The tool automatically maps common variations (e.g. "Job Number" → "Job #", "IQMS SN" → "Serial Number", "90 s" → "90", "ID" → "Initials").
